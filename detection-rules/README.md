# Custom Detection Rules

This folder contains custom Wazuh detection rules written as part of this lab.

## Rule Format
Wazuh rules are written in XML. Each rule has:
- A unique ID (100000+ for custom rules)
- A severity level (1-15)
- A description
- Logic that matches specific log patterns
