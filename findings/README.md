# Lab Findings & Investigation Notes

This folder documents security exercises, simulated incidents, and investigation notes from the home lab.

## Format
Each finding includes:
- **Scenario:** What was simulated or observed
- **Detection:** Whether Wazuh caught it and how
- **MITRE ATT&CK Mapping:** Which tactic/technique applies
- **Outcome:** What I learned, gaps found, tuning applied
